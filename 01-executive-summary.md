# 1. Executive Summary

## Course Vision

The **Enterprise Generative AI Engineer Course** is a production-first, job-ready program for the United States AI labor market (2026–2027). It trains engineers to build, deploy, evaluate, and operate LLM-powered systems at enterprise scale—not to pass academic exams or replicate tutorial demos.

## Market Context

- **78%+** of US organizations use AI in at least one business function (McKinsey, 2026).
- **AI Engineer** is LinkedIn’s #1 fastest-growing job in the US for 2026.
- AI job postings grew **~74% YoY**; Q1 2026 added 9,900+ new AI roles (55,374 total openings).
- Median **Generative AI Engineer** base salary: **$175K**; total comp ~**$254K** (US, 2026).
- **90%** of enterprises report critical AI skill shortages—the gap is production engineering, not awareness.

## What Makes This Course Different

| Shallow Bootcamp / YouTube | This Curriculum |
|---------------------------|-----------------|
| Chatbot in a notebook | RAG + agents + LLMOps in production |
| Single API call demos | Multi-tenant, observable, cost-optimized systems |
| LangChain copy-paste | Architecture, evaluation, security, governance |
| No deployment | Docker, K8s, CI/CD, cloud (AWS/Azure/GCP) |
| No evaluation | Ragas, DeepEval, promptfoo, regression suites |
| Theory-heavy ML | Applied GenAI: RAG, agents, copilots, workflows |

## Target Graduate Profile

An engineer who can:

1. Design and implement **RAG pipelines** with hybrid search, reranking, and semantic caching.
2. Build **agentic systems** (single and multi-agent) with tool calling, memory, and orchestration.
3. Operate **LLMOps**: tracing, monitoring, prompt versioning, guardrails, AI CI/CD.
4. Deploy on **cloud AI stacks** with security, RBAC, compliance, and cost controls.
5. Ship **full-stack AI products** (FastAPI + Next.js, streaming, multimodal).
6. Pass **technical interviews**: system design, architecture, coding, take-homes.

## Program Structure

- **12 curriculum modules** — 40+ topics, each with 14 structured fields
- **13 capstone-grade projects** — enterprise, startup, and consulting patterns
- **25+ job role mappings** — salary, skills, portfolio expectations
- **6- and 12-month roadmaps** — intensive and standard pacing
- **Enterprise readiness checklist** — hireability audit before job search

## Core Competency Stack

```
Python + FastAPI + TypeScript/Next.js
    → LLM APIs (OpenAI, Anthropic, Gemini)
    → RAG (vector DBs, hybrid search, reranking)
    → Agents (LangGraph, CrewAI, AutoGen, Semantic Kernel)
    → LLMOps (Langfuse, eval pipelines, guardrails)
    → Cloud (AWS Bedrock/SageMaker, Azure AI, Vertex AI)
    → Enterprise (security, governance, compliance)
    → Production (K8s, observability, cost optimization)
```

## Success Metrics

Graduates should demonstrate:

- **3–5 production-grade GitHub projects** with architecture diagrams and README runbooks
- **Measurable eval metrics** on RAG and agent systems (faithfulness, latency, cost/query)
- **Deployment evidence** (cloud URLs, Docker, CI badges)
- **Interview artifacts** (system design docs, take-home quality bar)

## Instructional Philosophy

1. **Build first, theory when needed** — math only where it improves engineering decisions.
2. **Every topic ties to a job role** — no orphan concepts.
3. **Enterprise defaults** — security, logging, and evaluation are day-one, not “phase 2.”
4. **Portfolio as curriculum** — projects are the credential.
