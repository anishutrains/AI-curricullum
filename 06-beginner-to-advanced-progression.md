# 6. Beginner to Advanced Progression

## Progression Ladder

```
Level 0: Software Developer (prerequisite)
    ↓
Level 1: AI Application Developer (Modules 01–03)
    ↓
Level 2: RAG / LLM Engineer (Module 04)
    ↓
Level 3: Agentic AI Engineer (Module 05)
    ↓
Level 4: Production AI Engineer (Modules 06, 08, 11)
    ↓
Level 5: Enterprise / Platform AI Engineer (Modules 07, 10)
    ↓
Level 6: AI Architect / Staff Engineer (Capstone + system design)
```

## Level Definitions

### Level 1 — AI Application Developer
**Can:** Call LLM APIs, build simple chat UIs, basic prompting  
**Cannot yet:** Production RAG, agents, LLMOps  
**Portfolio:** LLM API wrapper, prompt playground  
**Salary target:** $100K–$140K  

### Level 2 — RAG / LLM Engineer
**Can:** End-to-end RAG, vector DBs, chunking, basic eval  
**Cannot yet:** Multi-agent orchestration at scale  
**Portfolio:** Enterprise knowledge assistant with metrics  
**Salary target:** $140K–$175K  

### Level 3 — Agentic AI Engineer
**Can:** LangGraph agents, tool calling, multi-agent workflows  
**Cannot yet:** Full platform LLMOps, enterprise compliance programs  
**Portfolio:** Research agent, workflow automation  
**Salary target:** $155K–$190K  

### Level 4 — Production AI Engineer
**Can:** Deploy, monitor, evaluate, optimize cost/latency  
**Cannot yet:** Org-wide architecture standards  
**Portfolio:** LLMOps pipeline, deployed SaaS copilot  
**Salary target:** $170K–$210K  

### Level 5 — Enterprise / Platform AI Engineer
**Can:** Governance, security, multi-tenant cloud, K8s  
**Portfolio:** Compliance-ready system with ADRs  
**Salary target:** $185K–$230K  

### Level 6 — AI Architect
**Can:** System design, vendor strategy, team technical leadership  
**Portfolio:** Reference architectures, capstone at scale  
**Salary target:** $200K–$280K+ total comp
