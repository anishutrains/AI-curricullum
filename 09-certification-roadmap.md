# 9. Certification Roadmap

Certifications **supplement** the portfolio—they do not replace shipped projects. Pursue in parallel with Modules 07–08.

## Priority Tiers

### Tier 1 — High ROI for US AI Engineer Roles

| Certification | Provider | Align With Module | Est. Prep |
|---------------|----------|-------------------|-----------|
| **AWS Certified Machine Learning – Specialty** | AWS | 08 | 6–8 weeks |
| **Azure AI Engineer Associate (AI-102)** | Microsoft | 07, 08 | 4–6 weeks |
| **Google Professional Machine Learning Engineer** | GCP | 08 | 6–8 weeks |

*Pick **one** primary cloud aligned to your target employers.*

### Tier 2 — Valuable for Platform / Security Roles

| Certification | Provider | Align With |
|---------------|----------|------------|
| **CKA (Certified Kubernetes Administrator)** | CNCF | 08, 11 |
| **Terraform Associate** | HashiCorp | 11 |
| **AWS Solutions Architect Associate** | AWS | 08 (breadth) |

### Tier 3 — Specialized / Optional

| Certification | When Worth It |
|---------------|---------------|
| **NVIDIA NCA-GENL** | GPU/infrastructure-focused roles |
| **ISC2 CC (Certified in Cybersecurity)** | AI Security Engineer path |
| **PMI CAPM** | AI Consultant with program delivery |

## Not Recommended as Primary Credential

- Generic "AI prompt engineering" certificates without labs
- Vendor-agnostic theory-only ML certs without GenAI production content
- Coursera completion certificates alone (use for learning, not resume lead)

## 6-Month Certification Schedule (Intensive)

| Month | Activity |
|-------|----------|
| 1–2 | Modules 01–04; no cert yet |
| 3 | Start Azure AI-102 OR AWS ML Specialty study |
| 4 | Module 08 labs = hands-on cert prep |
| 5 | Take cloud AI exam |
| 6 | Optional CKA if Platform track |

## 12-Month Certification Schedule (Standard)

| Quarter | Certification |
|---------|---------------|
| Q1 | Modules 01–04 |
| Q2 | Azure AI-102 or AWS ML Specialty |
| Q3 | CKA or Terraform Associate |
| Q4 | Second cloud cert only if multi-cloud role target |
