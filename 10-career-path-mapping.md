# 10. Career Path Mapping

## Path A — Product AI Engineer (Startup / Big Tech)

```
Modules 01→02→03→04→05→06→09
Projects: RAG Assistant, Customer Support, Coding Copilot
Roles: Generative AI Engineer, AI Product Engineer, AI Copilot Engineer
Salary trajectory: $140K → $200K+ (3–5 years)
```

## Path B — Enterprise AI Engineer (Fortune 500 / Consulting)

```
Modules 01→04→06→07→08→10
Projects: Enterprise RAG, Healthcare/Legal, Integration copilot
Roles: Enterprise AI Engineer, AI Solutions Engineer, AI Consultant
Certs: Azure AI-102 or AWS ML Specialty
Salary: $150K → $210K
```

## Path C — Agentic AI Specialist

```
Modules 01→02→05→06→09
Projects: Research Agent, Multi-Agent Finance, Workflow Platform
Roles: Agentic AI Engineer, AI Workflow Engineer, AI Automation Engineer
Salary: $160K → $230K
```

## Path D — AI Platform / LLMOps

```
Modules 01→06→08→11
Projects: RAG + LLMOps pipeline, DevOps Assistant, K8s inference
Roles: LLMOps Engineer, AI Platform Engineer, AI Infrastructure Engineer
Certs: CKA, Terraform, cloud AI
Salary: $170K → $250K
```

## Path E — AI Security & Responsible AI

```
Modules 01→03→10→04 (RAG security angle)
Projects: Cybersecurity Analyst, guardrail-heavy copilot
Roles: AI Security Engineer, Responsible AI Engineer
Salary: $155K → $220K
```

## Path F — Freelance / Consulting

```
Modules 01→04→07→12
Projects: 2-week POC templates, case studies
Roles: AI Consultant, freelance AI Solutions Engineer
Income: $150–$300/hr senior; productized POC packages $15K–$50K
See: job-readiness/freelance-consulting-roadmap.md
```

## Path G — Remote-First International (US companies)

```
Same technical paths; add:
• Async communication portfolio (docs, Loom demos)
• US-timezone overlap 4+ hours
• Strong GitHub + written case studies
See: job-readiness/remote-job-strategy.md
```

## Decision Tree

```
Do you prefer building product features?
  YES → Path A
  NO → Do you prefer infra/ops?
    YES → Path D
    NO → Do you prefer client-facing work?
      YES → Path B or F
      NO → Are you most excited by agents?
        YES → Path C
        NO → Path E (security) or Path A (default)
```
