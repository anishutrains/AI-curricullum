# 13. Twelve-Month Standard Roadmap

**Commitment:** 12–15 hours/week  
**Outcome:** Senior-ready Generative AI Engineer portfolio + optional cloud cert

## Quarter 1 (Months 1–3) — Foundations & RAG

| Month | Modules | Projects |
|-------|---------|----------|
| 1 | 01, 02 (partial) | FastAPI starter + LLM router |
| 2 | 02, 03 | Prompt CI + DSPy lab |
| 3 | 04 | **Enterprise RAG Assistant** v1 |

**Q1 Exit:** Deployed RAG with basic eval metrics.

## Quarter 2 (Months 4–6) — Agents & LLMOps

| Month | Modules | Projects |
|-------|---------|----------|
| 4 | 04 advanced, 05 start | RAG + rerank + cache |
| 5 | 05 | **Research Agent** or **Customer Support** |
| 6 | 06 | LLMOps on all projects |

**Q2 Exit:** Agent + tracing + CI eval gates.

## Quarter 3 (Months 7–9) — Enterprise & Cloud

| Month | Modules | Projects |
|-------|---------|----------|
| 7 | 07 | Multi-tenant + compliance docs |
| 8 | 08 | Cloud deploy + cert study |
| 9 | 08, 09 | Full-stack UI + **second project** |

**Q3 Exit:** Cloud certification attempt; 2 production projects.

## Quarter 4 (Months 10–12) — Security, Capstone, Career

| Month | Modules | Projects |
|-------|---------|----------|
| 10 | 09, 10 | SaaS or workflow project |
| 11 | 10, 11 | Security hardening + ADRs |
| 12 | 12 | **Capstone** + job search |

**Q4 Exit:** 3–5 hire-ready repos; interview prep complete.
