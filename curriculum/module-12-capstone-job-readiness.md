## Topic 12.1: Capstone Project Execution

### 1. Topic Name
Industry Capstone — End-to-End Delivery

### 2. Why This Topic Matters
Hiring managers hire **one exceptional project** over ten tutorials. Capstone proves enterprise-ready execution.

### 3. Job Roles
All roles — choose vertical aligned to target job

### 4. Learning Objectives
- Deliver full-stack production AI system in 4–6 weeks
- Meet enterprise readiness checklist (Section 14)
- Present architecture and metrics to mock panel

### 5. Lessons/Subtopics
- Capstone selection rubric
- Sprint planning (4 × 1-week sprints)
- Architecture review checkpoint
- Security review checkpoint
- Eval gate before launch
- Demo video + technical blog post

### 6. Hands-on Labs
**Capstone build** — see [08-project-portfolio.md](../08-project-portfolio.md) for specs.

### 7. Enterprise Projects
Choose one:

| Project | Target Role |
|---------|-------------|
| Enterprise RAG Assistant | RAG / Enterprise AI |
| AI Customer Support System | Conversational AI |
| AI Coding Copilot | Copilot Engineer |
| Autonomous Research Agent | Agentic AI |
| Multi-Agent Finance Analyst | Applied AI / Finance |
| AI Healthcare Assistant | Enterprise + Compliance |
| AI Legal Document Analyzer | Applied AI |
| AI Cybersecurity Analyst | AI Security |
| Multimodal AI Search Engine | LLM / Product |
| AI Workflow Automation Platform | Workflow / Automation |

### 8. Portfolio Projects
Capstone = **flagship GitHub repo** with stars-worthy README.

### 9. Interview Preparation
- 10-minute capstone presentation script
- Deep-dive questions per architecture choice
- "What would you do differently at 10× scale?"

### 10. Real-world Use Cases
Direct substitute for 6–12 months junior industry experience when well-executed.

### 11. Tools & Technologies
Full stack from Modules 01–11

### 12. Common Mistakes
- Starting capstone before Module 04 complete
- No eval metrics in README
- Undeployed "localhost only" capstone

### 13. Industry Best Practices
- Write post-mortem blog within 2 weeks of finish
- Open-source non-proprietary components
- Pin demo video/GIF above fold in README

### 14. Capstone Deliverables

**Required package:**

- [ ] Public GitHub repo with MIT license
- [ ] Architecture diagram (C4 container)
- [ ] Deployed URL or video demo
- [ ] Eval metrics (faithfulness, latency, cost/query)
- [ ] SECURITY.md + COMPLIANCE.md (as applicable)
- [ ] Runbook for one failure scenario
- [ ] 1-page executive summary PDF

## Topic 12.2: Job Search & Interview Mastery

### 1. Topic Name
AI Job Search, Interviews & Career Launch

### 2. Why This Topic Matters
Strong engineers fail hires due to weak portfolios and system design—not weak coding.

### 3. Job Roles
All — transition to employment

### 4. Learning Objectives
- Optimize resume, LinkedIn, GitHub for AI roles
- Pass system design, coding, prompt, and take-home interviews
- Execute remote and consulting job strategies

### 5. Lessons/Subtopics
- AI resume format (impact metrics, not buzzwords)
- GitHub portfolio curation (pin 3 repos)
- LinkedIn headline and featured projects
- System design interview patterns
- Coding: Python async, data structures still required
- Prompt engineering interviews
- Take-home assignment strategy (time-box, document)
- Negotiation and leveling basics

### 6. Hands-on Labs
**Lab 12.1:** Mock system design with peer; record feedback.
**Lab 12.2:** Complete 1 take-home in 48h with eval README.

### 7. Enterprise Projects
N/A — personal career deliverables.

### 8. Portfolio Projects
Polish 3 repos to "hire-ready" standard using checklist Section 14.

### 9. Interview Preparation
See [15-interview-preparation-matrix.md](../15-interview-preparation-matrix.md) and [job-readiness/](../job-readiness/).

### 10. Real-world Use Cases
Job applications, referrals, conference networking.

### 11. Tools & Technologies
LinkedIn, GitHub, Excalidraw, Pramp/interviewing.io, levels.fyi

### 12. Common Mistakes
- Resume lists courses, not shipped projects
- No metrics on portfolio READMEs
- Overstudying ML theory vs system design

### 13. Industry Best Practices
- Apply to 10–15 targeted roles/week after portfolio ready
- Warm referrals via OSS and blog posts
- Negotiate total comp, not base only

### 14. Capstone Deliverables
- Final resume + LinkedIn
- 3 pinned GitHub repos
- Completed mock interviews (3 types)
- 30-day job search plan
