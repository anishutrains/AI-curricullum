## 30-Day Job Launch Plan

### Week 1 — Assets
- [ ] Finalize resume (1 page, metrics-led)
- [ ] Pin 3 GitHub repos; README polish
- [ ] LinkedIn headline + featured section
- [ ] Record 2-min capstone Loom

### Week 2 — Practice
- [ ] 2 system design mocks
- [ ] 6 coding problems
- [ ] Complete 1 practice take-home

### Week 3 — Outreach
- [ ] 15 targeted applications (not mass spray)
- [ ] 10 connection requests with personalized note
- [ ] 1 blog post or technical thread on capstone

### Week 4 — Interviews
- [ ] Track pipeline in spreadsheet
- [ ] Debrief after each interview within 24h
- [ ] Iterate README/answers based on feedback

## Target Application Ratio

| Channel | % Effort |
|---------|----------|
| Referrals / network | 40% |
| Direct company portals | 30% |
| Recruiters (specialized AI) | 20% |
| Job boards (LinkedIn, Wellfound) | 10% |
