# Freelance AI Consulting Roadmap

## Positioning

**Niche:** Production RAG + agent POCs for mid-market companies (50–500 employees)  
**Not:** "I do ChatGPT integrations" (commoditized)

## Service Packages

| Package | Duration | Price (US, 2026) | Deliverable |
|---------|----------|------------------|-------------|
| AI Discovery Workshop | 2 days | $5K–$10K | Readiness report + roadmap |
| RAG POC | 2–3 weeks | $15K–$35K | Deployed assistant + eval report |
| Agent Workflow POC | 3–4 weeks | $25K–$50K | LangGraph workflow + runbook |
| LLMOps Audit | 1 week | $10K–$20K | Gap analysis + CI eval template |

## Client Acquisition

1. **LinkedIn content** — case studies from course projects (fictionalized clients OK if labeled)
2. **Former employers** — "available for advisory hours"
3. **Partner channels** — web dev agencies without AI bench
4. **Platforms** — Toptal, Braintrust (after 2 case studies)

## Legal & Ops Basics

- LLC formation (consult attorney for your state)
- MSA + SOW templates (limit liability, define IP ownership)
- Errors & omissions insurance for consulting
- Standard: client owns custom code; you retain generic frameworks

## Delivery Process

```
Discovery call → Written proposal → 50% deposit → Sprint → Demo → Final invoice → 30-day support window
```

## Upsell Path

POC → Production hardening contract ($50K–$150K) → retainer for LLMOps

## Portfolio for Consulting

- 2-page case study PDF per project
- ROI calculator spreadsheet (hours saved × rate)
- Reference architecture diagrams
